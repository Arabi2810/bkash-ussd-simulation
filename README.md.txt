# bKash USSD Service Simulation (*247#) in C

## 📌 About the Project

This project is a console-based simulation of the bKash USSD service (*247#), implemented using the C programming language.

It replicates the menu-driven interaction flow used in mobile banking USSD systems, where users navigate different financial services by entering numeric options.

This is not a real banking system. It is a learning-based simulation designed to understand how structured programs, user input handling, and real-world service flows can be implemented using C programming.

---

## ⚙️ Features

The system simulates a complete USSD-based mobile banking service, including:

- Send Money  
- Send Money to Non-bKash User  
- Mobile Recharge  
- Payment 
- Cash Out  
- Pay Bill  
- Microfinance 
- Download App
- My bKash  
- Reset PIN

All features are accessible through a structured menu system similar to real USSD navigation.

---

## 🧠 Concepts Used

This project is built using core C programming concepts:

- Functions (for modular design of features)  
- Structures (for handling user-related data where needed)  
- Conditional statements (if-else logic)  
- Switch-case (for menu navigation)  
- Loops (for continuous interaction flow)  
- Variables for basic data handling  

These concepts helped in building a structured and interactive USSD simulation system.

---

## 🧩 How It Works

The program follows a hierarchical menu system similar to real USSD services:

1. The user is shown the main menu (simulating *247# interface)  
2. The user selects an option using numeric input  
3. Each option leads to a specific service or sub-menu  
4. The system processes the input and displays the result  
5. The menu continues until the user exits the program
